Get started

Build a local autonomous storage lab.

ArgosFS is a Rust project with a management CLI, FUSE mount frontend, retained validation data, and artifact-evaluation scripts.

01

Install dependencies

sudo apt-get install -y build-essential pkg-config libfuse3-dev fuse3 smartmontools
02

Build and test

cargo build --release
cargo test
cargo clippy --all-targets --all-features -- -D warnings
03

Create a volume

argosfs mkfs /var/lib/argosfs/root --disks 6 --k 4 --m 2 --compression zstd
04

Mount and inspect

argosfs mount /var/lib/argosfs/root /mnt/argos-root --foreground
argosfs health /var/lib/argosfs/root --json
