ArgosFS

Self-driving erasure-coded storage for root filesystem experiments.

cargo build --release scripts/experiments/run_all.sh --quick

Control Loop

01

Observe

Probe devices, SMART fields, latency, capacity, and workload signals.

02

Plan

Score drain, scrub, rebalance, and observe actions with explicit safety checks.

03

Act

Rewrite shards across diverse failure domains with bounded background budgets.

04

Verify

Run fsck verification and downgrade automation when the system gets uncertain.

Compatibility Surface

FUSE operations include POSIX metadata, ACL xattrs, rename flags, sticky directories, non-UTF-8 filenames, and fsync durability hooks.

argosfs autopilot ROOT --dry-run --explain --json
argosfs verify-journal ROOT
